# Heart Disease Data Cleaning and Visualization Project

## Project Overview

This project focuses on cleaning, preprocessing, and visualizing a heart disease dataset to uncover meaningful patterns and insights. The analysis includes handling duplicates, exploring feature distributions, identifying correlations, and generating visual reports.

## Objectives

* Perform data cleaning and preprocessing
* Analyze demographic and clinical features
* Identify relationships between variables
* Create informative visualizations
* Extract actionable insights from healthcare data

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Kaggle Dataset

https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

## Key Findings

* Most patients are between 50–65 years old.
* Male patients are more prevalent in the dataset.
* Maximum Heart Rate and Chest Pain Type show strong relationships with heart disease.
* Cholesterol and Resting Blood Pressure exhibit weaker individual associations.
* Correlation analysis highlights important clinical factors affecting heart disease risk.

## Conclusion

Data visualization and exploratory analysis provided valuable insights into patient characteristics and factors associated with heart disease. These findings can support future predictive modeling and healthcare analytics applications.
